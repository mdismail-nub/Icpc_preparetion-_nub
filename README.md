# ICPC Preparation Tracker

A collaborative repository for tracking the progress of students preparing for the International Collegiate Programming Contest (ICPC) and improving problem-solving skills through consistent practice.

## 🎯 Objective

This repository is created to:

- Track the daily/weekly practice progress of members
- Maintain a collection of solved problems with explanations
- Encourage consistency and accountability
- Help beginners learn from others’ solutions
- Build a strong competitive programming community

## 📌 Contribution Rules

Every participant should upload:

- Problem link
- Problem name
- Platform name
- Solution code
- Optional explanation/thought process

You can solve problems from platforms like:

- Codeforces
- AtCoder
- CSES
- LeetCode
- CodeChef
- UVa
- SPOJ
- HackerRank
- LightOJ
- Others
